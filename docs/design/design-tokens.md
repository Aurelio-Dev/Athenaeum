# Athenaeum Design Tokens

Tokens iniciais extraidos do prototipo do Claude Design para a Library View.

## Superficies

| Token | Valor | Uso |
| --- | --- | --- |
| `surface.app` | `#FAFAFA` | Fundo principal da aplicacao |
| `surface.panel` | `#FFFFFF` | Header e paineis claros |
| `surface.card` | `#FFFFFF` | Cards de documentos |
| `surface.muted` | `#F3F4F6` | Contadores, controles segmentados |
| `surface.subtle` | `#EEF0F3` | Trilho de progresso |

## Tags de assunto

As tags de assunto usam apenas Violet, Indigo, Blue, Teal, Rose e Amber. Green, Slate e Red ficam reservados para status.

| Assunto | Tom | Fundo | Texto |
| --- | --- | --- | --- |
| Machine Learning | Violet | `#EDE9FE` | `#5B21B6` |
| Systems / Infra | Indigo | `#E0E7FF` | `#3730A3` |
| NLP | Blue | `#DBEAFE` | `#1D4ED8` |
| Computer Vision | Teal | `#CCFBF1` | `#0F766E` |
| Theory / Math | Rose | `#FFE4E6` | `#BE123C` |
| AI Safety / Ethics | Amber | `#FEF3C7` | `#B45309` |
